## Installation:
All builds and downloads are available [here](https://github.com/muunapp/muun/releases) in the repo releases section.

## Development

Installation:
```
$ git clone https://github.com/muunapp/muun.git
$ cd muun
```

Running:
```
$ npm start
```

Packaging:
```
$ npm run package
```
